# Landing hero video

Drop your video here as **`hero.mp4`** (H.264 MP4 — safest cross-browser) and it plays
automatically muted + looped behind the headline on `/`.

- Optional: `hero.webm` (added as a second `<source>`), and `hero-poster.jpg` (first frame
  shown before the video loads / if autoplay is blocked).
- If no file is present, the hero falls back to an animated gradient — nothing breaks.
- Recommended: a short (5–15s) loop, ≥1920px wide, muted, dark/moody so the white headline
  stays legible (a scrim overlay already darkens it).

To grab the Pinterest clip: open the pin, use a Pinterest/Instagram downloader or screen-record
it, export to MP4, and save it here as `hero.mp4`. Note it's third-party content — fine for a
hackathon demo, but don't ship it to production without rights.
